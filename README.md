# Setup deste workspace

* Instalar ROS 2
* Executar arquivo "build_ros.sh" com um dos comando abaixo
  * **OBS:** Deve ser executado na pasta raiz do projeto - "ros-foxy-tutorials"
  * ```bash build_ros.sh```
  * ```sh build_ros.sh```
* Em seguida referencie o workspace
  * ```. install/local_setup.bash```
